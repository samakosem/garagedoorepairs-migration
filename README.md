# garagedoorepairs.com SEO Migration Project

This project is for rebuilding https://garagedoorepairs.com/ from WordPress into a clean, high-performance coded website.

This is not a normal redesign project.

This is a critical SEO migration project.

## Main Objective

Rebuild the website with a premium, modern, unique design while protecting and improving the existing SEO value.

The goal is to rank stronger for:

Garage Door Repair Los Angeles

## Important

Do not delete existing pages.
Do not change important URLs.
Do not rename routes.
Do not remove SEO content.
Do not launch without a redirect map.
Do not launch without migration QA.

## Project Folders

- docs/  
  SEO migration rules and strategy documents.

- exports/  
  URL inventory, Google Search Console exports, indexed pages, and traffic data.

- backlinks/  
  Ahrefs or backlink exports by target URL.

- crawl/  
  Screaming Frog crawl files and technical crawl reports.

- redirect-map/  
  Final 301 redirect map before launch.

- current-site/  
  Screenshots, HTML exports, content backups, and notes from the existing WordPress site.

- new-site/  
  The new Next.js website will be built here.

- assets/  
  Logos, photos, icons, brand assets, and design materials.

- screenshots/  
  Before and after screenshots for QA.

## Build Direction

The new website should be:

- SEO-first
- Fast
- Mobile-first
- Premium
- Unique
- Conversion-focused
- Local Los Angeles focused
- Built with clean Next.js code

## Main Conversion Goals

- Phone calls
- Emergency garage door repair leads
- Quote requests
- Contact form submissions

## SEO Protection Workflow

1. Full URL inventory
2. Google Search Console Pages export
3. Backlinks by target URL
4. Screaming Frog crawl
5. Redirect Map
6. Keep / Rebuild / 301 / Noindex / Delete decision for every URL
7. Build new SEO structure
8. Technical SEO implementation
9. Migration QA before launch
10. Monitor Google Search Console after launch

## Final Rule

Protect the existing SEO asset first.

Improve design, speed, UX, content, and conversions second.

Never sacrifice indexed pages, backlinks, or important URLs for design convenience.
